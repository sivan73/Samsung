# Samsung
Samsung Innovation Campus
Personal Notebook used for handson python training in Samsung Innovation Campus 

This repository is a personal Python learning workspace created during the Samsung Innovation Campus training program.

## Purpose
It contains hands-on practice files, exercises, datasets, and small projects used to learn core Python concepts and data science workflows.

## Topics covered
- Python basics and OOP concepts
- Data structures and algorithms
- Pandas, NumPy, and data analysis
- Visualization with Matplotlib and Seaborn
- CSV/JSON-based data exploration
- Dashboard-style outputs and simple project examples

## Repository contents
- Daily practice scripts and notebooks-style exercises
- CSV and HTML files used for analysis and visualization
- Small example projects and custom Python modules

This repo is mainly intended for learning, experimentation, and practice.
